# Microsoft Security Operations Learning Path

This repository documents a structured learning path for Microsoft Security Operations, covering topics from Identity and Access Administration to Security Engineering and Support Engineering.

It is designed as a personal study guide, reference hub, and career development resource for understanding Microsoft security roles, tools, workflows, and best practices.

## Purpose

The goal of this repository is to:

- Track my learning progress in Microsoft security operations
- Organize study materials in one place
- Build practical knowledge across security roles
- Create a reference for future revision and preparation

## Learning Areas

This repository covers the following core areas:

### Identity & Access Management
- Microsoft Entra ID (Azure AD)
- Authentication & Authorization
- Role-Based Access Control (RBAC)
- Conditional Access
- Privileged Identity Management (PIM)

### Windows Server & On-Prem Identity
- Active Directory Domain Services (AD DS)
- Domain Controllers & Forests
- Group Policy (GPO)
- DNS & DHCP fundamentals
- Hybrid Identity (AD DS + Entra ID)
- Azure AD Connect

### Security Operations
- Security monitoring and alerting
- Incident detection and response
- Log analysis and investigation
- Threat hunting fundamentals

### Security Engineering
- Microsoft Defender suite
- Microsoft Sentinel (SIEM/SOAR)
- Endpoint security
- Identity protection
- Security architecture and design

### Security Support Engineering
- Troubleshooting identity issues
- Resolving access and authentication problems
- Diagnosing security alerts and incidents
- Supporting enterprise security environments

---

## Learning Path

This repository follows a structured progression:

1. Windows Server Fundamentals (AD DS, DNS, GPO)
2. Identity & Access Administrator
3. Hybrid Identity (On-prem + Cloud)
4. Security Operations (SOC fundamentals)
5. Security Engineering
6. Security Support Engineering
